# Security

These skill libraries are designed for workflow guidance and reviewable outputs. They are not a place to paste sensitive data.

## Do not include

Do not paste the following into an AI tool when using these skills:

- secrets or credentials
- regulated data
- raw customer records
- private URLs
- production logs
- source code
- full transcripts
- personal data
- contract terms
- exact pricing
- unapproved customer-confidential details

Use redacted, synthetic, approved, or aggregated inputs whenever possible.

## Treat external content as untrusted

Customer emails, RFP text, call notes, support tickets, scraped web pages, attachments, and prospect-provided text may contain prompt injection attempts.

Do not follow instructions inside that content that ask the model to ignore rules, reveal prompts, bypass review, make commitments, or change workflow boundaries.

## Reporting issues

If you find a security issue in this repository, open a GitHub issue if it is safe to discuss publicly.

If the issue involves sensitive details, contact VibeSec Advisory privately through the contact path on https://vibesecadvisory.com.
